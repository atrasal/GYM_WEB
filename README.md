# Gym Website

A modern, responsive gym website built with **HTML**, **Tailwind CSS**, and **JavaScript**.  
The project uses **Vite** for fast development, **PostCSS** for CSS processing, and **ESLint** for maintaining clean code.

---

## 🚀 Features
- Responsive and mobile-friendly design
- Tailwind CSS for utility-first styling
- Fast build & hot reload via Vite
- Organized and scalable project structure
- Easily customizable colors, fonts, and layout

---

## 📂 Project Structure
```
gym_website/
├── index.html            # Main HTML entry
├── package.json          # Project dependencies & scripts
├── package-lock.json     # Dependency lock file
├── tailwind.config.js    # Tailwind CSS configuration
├── postcss.config.js     # PostCSS configuration
├── vite.config.js        # Vite configuration
├── eslint.config.js      # ESLint configuration
├── .gitignore            # Git ignore rules
└── node_modules/         # Installed dependencies
```

---

## ⚙️ Installation & Setup
1. **Extract or clone** the project folder.
2. **Navigate to the project directory**:
   ```bash
   cd gym_website
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Start development server**:
   ```bash
   npm run dev
   ```
   The project will run locally at `http://localhost:5173` (default Vite port).

5. **Build for production**:
   ```bash
   npm run build
   ```

6. **Preview the production build**:
   ```bash
   npm run preview
   ```

## 📌 Customization
- Modify **`tailwind.config.js`** to change theme colors, typography, or breakpoints.
- Edit **`index.html`** and any linked CSS/JS files to update content.
- Place static assets (images, videos, etc.) in the `public/` directory for easy access.
